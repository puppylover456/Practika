# Описание изменений в проекте

## 1. WorkshopManager.cs (новый файл)
**Назначение:** Центральный менеджер магазина - управляет материалами, наборами, клиентами и выручкой.

**Реализованные методы:**
- `AddMaterial()` / `AddKit()` - добавление в каталог
- `GetAllMaterials()` / `GetAllKits()` - получение всех позиций
- `FindMaterialsByCraftType()` - поиск материалов по виду творчества
- `RecommendKitsForCustomer()` - рекомендация наборов по уровню навыков клиента
- `RecordSale()` / `GetTotalRevenue()` - учёт выручки
- `FindCustomerByEmail()` / `FindCustomerByPhone()` - поиск клиента
- `RegisterCustomer()` - регистрация нового клиента
- `GetLowStockMaterials()` - материалы с низким остатком
- `GetPopularCraftTypes()` - статистика популярных видов творчества

---

## 2. Customer.cs
**Назначение:** Класс клиента магазина с проектами и списком желаний.

**Добавлено:**
- Свойство `SkillLevel` (новичок/любитель/профи)
- Свойство `Interests` для доступа к интересам
- Метод `StartProject()` - создание нового проекта
- Метод `AddMaterialToProject()` - добавление материала в проект
- Метод `AddKitToProject()` - добавление набора в проект
- Метод `FinishProject()` - завершение проекта
- Метод `AddToWishList()` - добавление в список желаний
- Метод `AddInterest()` - добавление интереса
- Метод `GetActiveProjects()` - получение активных проектов
- Метод `GetAllProjects()` - получение всех проектов
- В `ShowCustomerInfo()` добавлен вывод уровня навыков

---

## 3. CraftMaterial.cs
**Назначение:** Класс материала для творчества.

**Добавлено:**
- Свойство `CraftType` (вид творчества: шитье, вязание, вышивка и т.д.)
- В конструкторе: проверка цены (если < 0, устанавливается 10)
- В конструкторе: сохранение `CraftType`
- Метод `ToString()` - форматированный вывод: "Название цвет (вид) - цена руб/единица (остаток единица)"

---

## 4. CraftKit.cs
**Назначение:** Класс набора для творчества.

**Добавлено:**
- Свойство `AgeRecommendation` (6+, 12+, 18+)
- Метод `AddMaterial()` - добавление материала в набор
- Метод `AddRequiredTool()` - добавление инструмента
- Метод `CalculateKitPrice()` - расчёт стоимости (материалы + 20% наценка)
- Метод `IsKitAvailable()` - проверка доступности всех материалов
- Метод `GetMissingMaterials()` - список недостающих материалов
- Метод `GetMaterials()` - получение словаря материалов
- В `ShowKitInfo()` добавлен вывод возрастной рекомендации

---

## 5. StoreMenu.cs
**Назначение:** Консольное меню и бизнес-логика магазина.

**Реализовано:**
- `InitializeStoreData()` - добавлены материалы в наборы, созданы дополнительные наборы
- `ShowMaterialsByCategory()` - вывод материалов, сгруппированных по видам творчества
- `ShowCraftKits()` - вывод наборов, сгруппированных по сложности
- `ProvideProjectConsultation()` - полный цикл консультации: регистрация/поиск клиента, подбор материалов и наборов, создание проекта
- `ProcessOrder()` - оформление заказа: выбор проекта, добавление материалов/наборов, расчёт стоимости, списание остатков, учёт выручки
- `ShowWorkshopsAndTips()` - мастер-классы, рекомендации по материалам, советы по экономии
- `ShowWorkshopStats()` - статистика: выручка, клиенты, популярные виды творчества, материалы с низким остатком
